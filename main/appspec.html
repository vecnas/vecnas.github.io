Единый реестр приложения / Application Specification

Базовое приложение описано в одной js переменной, динамика добавляется скриптами. Это значительно упрощает документирование, анализ, поддержку и изменение имеющегося приложения. Также обеспечивается посредничество для инструментов автозавершения при кодировании 

  var myApp = {
    id: "myApp",
    views: {
      main: {}
    }
  };

  jiant.bindUi(myApp)
