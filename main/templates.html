Шаблоны / Templates

Шаблоны необходимы при генерации содержимого. Их содержимое также проверяется и связывается с реализацией при старте приложения 

  templates: {
    tm: {
      elem: jiant.label
    }
  }
И затем 

  jiant.onUiBound(app, function($, app) {
    var elem = app.templates.tm.parseTemplate()
  })
