Виджеты / Views

Виджеты и их элементы объявляются в секции views и автоматически привязываются к html реализации 

  views: {
    navbar: {
      mainCtl: jiant.ctl,
      helpCtl: jiant.ctl
    }
  }
Префикс используется для уникальной изоляции html кода разных приложений. В приведенном примере используем префикс "_" 

  <div id="_navbar">
    <button class="_mainCtl">Main</button>
    <a class="_helpCtl">Help</a>
  </div>
Виджет привязывается к реализации по идентификатору, элементы виджета по классу
