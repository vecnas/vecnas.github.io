Jiant - JIANT - Javascript Interface Abstract Notation

Цель данной библиотеки - максимально повысить управляемость и простоту разработки крупных высокодинамичных аякс проектов на Javascript. Дополнительная цель - уменьшение повторяющегося кода путем переиспользования алгоритмов и автогенерации кода.

Jiant предоставляет возможность использования автоматических инструментов современных средств разработки. Это достигается путем замены строковых значений на имена переменных.

Дополнительно Jiant осуществляет проверку связности скрипта и html документа насколько возможно рано - на старте приложения вместо момента использования какого-либо элемента, что значительно повышает качество кода и уменьшает количество ошибок.
