Состояния / States

Сохранение параметров в адресной строке браузера, навигация и восстановление при обновлении окна браузера. Код работы с состояниями генерируется на основе объявления следующего вида 

  states: {
    main: {
      go: function(param) {},
      start: function(cb) {},
      end: function(cb) {}
    }
  }
Дальше остается только использовать 

  app.states.main.start(function(param) {
    alert("we in a main state now, with parameter " + param)
  })
