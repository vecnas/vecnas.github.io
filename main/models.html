Модель данных / Models

Интерфейс доступа к общим данным приложения с автогенерацией функций доступа и подпиской на события изменения 

  models: {
    tag: {
      add: function(arr) {},
      all: function() {},
      findById: function(id) {},
      id: function(id) {},
      name: function(name) {}
    }
  }
Весь код для объявления выше автогенерируется, остается пользоваться 

  var tag12 = app.models.tag.findById(12);
  tag12.name("new name"); // установить значение
  tag12.name(); // получить значение
  tag12.name.on(function(tag, name) {
    ...
  }) // подписываемся на изменения
