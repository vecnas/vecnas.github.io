Зависимости / Logic

Программная логика может быть абстрагирована посредством объявления интерфейса 

  logic: {
    sharedAppMath: {
      cos2tan3: function(rad) {}
    }
  }
Использующий ее код объявляет зависимость 

  jiant.onUiBound(app, ["sharedAppMath"], function($, app) {}]
И вызывается только когда интерфейс будет кем-нибудь реализован 

  jiant.onUiBound(app, function($, app) {
    app.logic.sharedAppMath.implement({
      cos2tan3: function(rad) {return someValue}
    })
  })
