Чеклист

  1. Создаем спецификацию приложения, включая нужные возможности и идентификатор 
    
      var myApp = {
        id: "myApp",
        appPrefix: "_",
        views: {
          main: {
            title: jiant.label
          }
        }
      }
  2. Реализуем в html: 
    
      <div id="_main">
        <div class="_title"></div>
      </div>
  3. Вызываем bindUi (почитайте документацию для списка возможных параметров) 
    
      $(function () {
        jiant.bindUi(myApp, true); // true - включает режим разработки, больше сообщений в логи и алерты об ошибках
      });
  4. Добавляем логику работы через анонимные плагины 
    
      jiant.onUiBound(myApp, function($, app) {
        app.views.main.title.html("Application started")
      });
