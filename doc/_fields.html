Jiant поля

Список открытых переменных для конфигурирования jiant со значениями по умолчанию: 

    AJAX_PREFIX: "",
    AJAX_SUFFIX: "",
    DEV_MODE: false,
    PAGER_RADIUS: 6,
    isMSIE: eval("/*@cc_on!@*/!1"),
  • AJAX_PREFIX, AJAX_SUFFIX - префикс и суффикс для серверных вызовов. Смотрите раздел Ajax для деталей. Могут выставляться глобально или для каждого приложения отдельно, через ajaxPrefix, ajaxSuffix в переменной определения приложения.
  • STATE_EXTERNAL_BASE - используется для навигации на внешний URL при смене состояния приложения. Может быть выставлено глобально или для каждого приложения отдельно, через stateExternalBase в определении приложения
  • DEV_MODE - включает функцию logInfo. Печатает больше информации и добавляет полезную для разработки функциональность. См. раздел Режим разработки для деталей.
  • DEBUG_MODE - включает режим тщательной отладки для разбирательства с наиболее сложными взаимодействиями при работе с событиями, состояниями и серверными вызовами. Содержит три поля. См. Тщательную отладку для деталей
  • PAGER_RADIUS - радиус генерации навигации для пейджера. См. Типы элементов для деталей
  • isMSIE - флаг для определения является ли текущий браузер Internet Explorer. Использует хак условной компиляции. Работает до сегодняшнего дня. Проверено в IE 8, 10. Вызов eval() защищает от очистки минификацией.
