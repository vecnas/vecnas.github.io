Использование Jiant

Jiant обеспечивает привязку и автоматическую реализацию для виджетов, шаблонов, серверных вызовов, шины событий, моделей данных, абстракций программной логики, приложений с состояниями (hash строка браузера).

Каждая возможность может быть использована отдельно, между ними нет зависимостей. Читайте соответствующие разделы.

Чтобы использовать Jiant - создайте спецификацию приложения и затем вызовите jiant.bindUi
