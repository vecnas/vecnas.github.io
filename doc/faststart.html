Детальный пример

  1. Создаем определение приложения в виде JSON: 
    
            var myapp = myapp || (function(jiant) {
                var container = jiant.container,
                ctl = jiant.ctl,
                label = jiant.label;
                return {

                    id: "myapp",

                    ajax: {
                        getData: function(dataId, cb) {}
                    },

                    views: {
                        mainLayoutView: {
                            contentContainer: container,
                            menuContainer: container,
                            title: label
                        },
                        mainMenuView: {}
                    },

                    templates: {
                        tmMenuItem: {
                            control: ctl
                        }
                    },

                };
            })(jiant);
    Никакого html/jscript на этом этапе - только чистая абстракция приложения - это важно для успешного анализа.
  2. Реализуем каждый виджет и шаблон в виде html: 
    
            <div id="_mainLayoutView">
                <h1 class="_title"></h1>
                <div class="_menuContainer"></div>
                <div class="_contentContainer"></div>
            </div>
    или 
    
            <ul id="_mainMenuView"></ul>
    или, шаблон: 
    
            <div id="_tmMenuItem">
                <li>
                    <a class="_control">!!label!!</a>
                </li>
            </div>
  3. На событие document.ready вызываем 
    
            jiant.bindUi("_", myapp);
  4. Реализуем динамическую логику работы в отдельных "плагинах": 
    
            jiant.onUiBound(myapp, [], function($, app) {
                var main = app.views.mainLayoutView,
                menu = app.views.mainMenuView,
                tmMenuItem = app.templates.tmMenuItem;

                main.menuContainer.append(menu);
                app.ajax.getData(someDataIdHere, function(serverData) {
                    $.each(serverData.menuItems, function(idx, menuItem) {
                        var menuElem = tmMenuItem.parseTemplate({label: menuItem.labelText});
                        menu.append(menuElem);
                        menuElem.control.click(function() {
                            alert("Where we are go today?");
                        });
                    });
                });
            });
