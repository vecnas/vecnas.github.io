Тщательная отладка

jiant.DEBUG_MODE содержит четыре поля, все по умолчанию выключены. Для тщательной отладки - нужно установить требуемые до вызова bindUi следующим образом: 

    jiant.DEBUG_MODE.ajax = 1;
    jiant.DEBUG_MODE.states = 1;
    jiant.DEBUG_MODE.events = 1;
    jiant.DEBUG_MODE.data = 1;
Другой вариант - добавить в урл (в любое место) строки 

    jiant.debug_events
    jiant.debug_states
    jiant.debug_ajax
    jiant.debug_data

Логгирование отладки

Когда любой из отладчиков включен, добавляется большой объем сообщений в логи - переходы состояний, подписки и вызовы событий с полным стеком вызова, и так далее. Используйте эти флаги для полного понимания как работает приложение или почему оно не работает как ожидается. В будущем эти флаги возможно будут убраны, они служили для отладки самого jiant на ранней стадии жизни.
