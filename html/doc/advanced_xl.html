Использование XL

XL - eXtra Logic - предоставляет готовые алгоритмические решения некоторых наиболее общих задач. Детальная документация по XL здесь.

Список с выбором

Задача - отрисовать набор кнопок, одна из которых будет выбираться (поведение аналогичное элементу radio). 

<div class="app-actions _dataSet">
  <button type="button" class="btn active">Today</button>
  <button type="button" class="btn">Yesterday</button>
  <button type="button" class="btn">7 days</button>
  <button type="button" class="btn">15 days</button>
</div>
Задача решается вызовами двух функций XL: Представление объектов списком и Псевдо селектор. Первая отрисует заданный список в контейнере, вторая добавит к нему необходимое поведение, например (при соответствующем именовании виджета и его элементов): 

  const select = jiant.xl.pseudoSelect([], [], myCb, 0, "active"]);
  jiant.xl.renderList([{class: 'btn', name: "Today"}, ...], app.v.mainDashboard.dataSet, app.t.dataButton)();
Элементы в select можно добавлять разными способами, в зависимости от вкуса - по мере отрисовки renderList, используя коллбэк (4й аргумент, perItemCb), либо после отрисовки динамически достать все необходимые элементы. Первый кажется более производительным.
