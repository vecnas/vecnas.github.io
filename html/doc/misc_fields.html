Поля jiant.

Jiant содержит несколько полей, которые можно разбить по смыслу по нескольким категориям

Глобальные настройки

AJAX_PREFIX Глобальная настройка префикса при формировании урла для аякс запросов, см. про аякс
AJAX_SUFFIX Глобальная настройка суффикса при формировании урла для аякс запросов, см. про аякс
DEV_MODE Включает режим разработки, в котором выдается больше отладочной информации
PAGER_RADIUS Количество показываемых страниц от текущей в элементе jiant.pager
LIB_LOAD_TIMEOUT Время загрузки внешней библиотеки, используется в библиотечных модулях, по умолчанию равен 5000 (5 секунд)
isMSIE Флаг, определяющий является ли текущий браузер Internet Explorer'ом
STATE_EXTERNAL_BASE Префикс состояний, см. про состояния
Например, 

    jiant.DEV_MODE = true;
    jiant.LIB_LOAD_TIMEOUT = 15000;
    jiant.AJAX_SUFFIX = ".php";

Типы элементов виджетов

Сюда входят поля типа jiant.pager, jiant.input, jiant.label и тд. Все типы описаны в секции про элементы виджетов
