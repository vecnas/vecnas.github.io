Введение

Jiant представляет собой фреймворк для разработки приложений на javascript, включающий несколько полезных инструментов.

На этом сайте находится документация по фреймворку Jiant, с работающими примерами и лучшими практиками, а также рецептами решения типовых задач при разработке фронтенд приложений на javascript.

Jiant обеспечивает привязку и автоматическую реализацию для
  • виджетов
  • шаблонов
  • серверных вызовов, ajax
  • шины событий, event bus
  • моделей данных
  • приложений с состояниями (hash строка браузера)
  • семафоров состояния
  • локализации
А также механизмы
  • модульной организации приложения
  • абстракций программной логики
  • организации зависимостей между кодом приложения

Каждая возможность может быть использована отдельно.
