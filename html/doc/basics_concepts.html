Концепции Jiant

Приложение на javascript задается через переменную определения приложения. Чтобы использовать Jiant - создайте спецификацию приложения и затем вызовите jiant.bindUi. После этого определенные в спецификации функции будет автоматически реализованы и привязаны к объектам на странице.

Большая часть приложения описывается пустыми функциями, которые затем получают реализацию от Jiant. Кстати, вы не можете сделать ошибку в том коде который не пишете.

Например, если определить модель 
  user: {
      jRepo: {}
      firstName: function(val) {},
      lastName: function(val) {},
  }
то после вызова bindUi, для этих методов будет создана реализация, а в секцию jRepo будут добавлены методы для работы с репозиторием моделей: 
  user: {
      jRepo: {
          all: ...
          add: ...
          remove: ...
Для виджетов, определенных в секции views, осуществляется поиск соответствующих элементов DOM страницы и их привязка. Это дает проверку кода на старте приложения вместо времени выполнения.

Переменная определения приложения записывается в формате JSON. Она включает следующие основные секции: 
    var appRoot = {
        id: "...",
        modules: "...",
        views: ...,
        templates: ...,
        ajax: ...,
        events: ...,
        states: ....,
        logic: ....,
        models: ....
    };
